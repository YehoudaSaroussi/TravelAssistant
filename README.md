## Setup and Installation

### Prerequisites

- JDK 17 or later
- Maven 3.6 or later

### Running Locally

1. Clone the repository

   ```
   git clone https://github.com/YOUR_USERNAME/travel-assistant.git
   cd travel-assistant
   ```

2. Build the project

   ```
   mvn clean install
   ```

3. Run the application

   ```
   mvn spring-boot:run
   ```

4. Access the application at http://localhost:8080
